## How to Work With Us

### Talk to anyone directly

```
Ctrl+Shift+0  Pim     "I need something but not sure what"
Ctrl+Shift+1  Bank    "Should we do X or Y?"
Ctrl+Shift+2  Fern    "Check this / debug this"
Ctrl+Shift+3  Mint    "Write Verilog for X"
Ctrl+Shift+4  Ohm     "Design circuit for X"
Ctrl+Shift+5  Bam     "Write tool/program for X"
Ctrl+Shift+6  Noon    "Write lab/docs for X"
```

### Or just tell Pim

Switch to Pim (Ctrl+Shift+0) and describe what you want. She'll figure out who handles what and in what order.

### We'll be frank with you

If anyone sees a problem — design is too complex, timeline is tight, something doesn't make sense — they'll tell you directly. We believe honest feedback is respect.

You can ask anyone "what do you really think?" and get a real answer.

---

## How We Work Together

```
You → Pim (decomposes) → Specialists (build) → Fern (verifies) → Done
```

- **Nobody verifies their own work.** Mint/Ohm/Bam build, Fern checks.
- **Natural pairs:** Mint+Fern, Ohm+Bank, Bam+Noon.
- **Cover each other's blind spots** — no ego, just teamwork.

## Quick Start

### Use in a project

```bash
cp -r JOJOCAFE-Org/.kiro /path/to/your/project/
cp JOJOCAFE-Org/{TEAM_MEMORY.md,PROJECT_STATE.md} /path/to/your/project/
```

### Portability

Everything is files in git. Clone → copy `.kiro/` → done. Works on any machine with Kiro.
