# RV8-GR Hardware Labs — สร้าง CPU จากศูนย์

**10 Lab sheets สำหรับน้อง ม.ต้น — ต่อวงจรทีละขั้น จนได้ CPU ที่ทำงานจริง**

---

## วิธีใช้

1. ทำทีละ Lab **ตามลำดับ** (ห้ามข้าม!)
2. ทุก Lab มี: อุปกรณ์ → ต่อวงจร → ทดสอบ → ✅ ผ่านแล้วค่อยไป Lab ถัดไป
3. ใช้ single-step clock (กดปุ่ม) ทดสอบ ก่อนใส่ crystal
4. ดู LED เพื่อเช็คว่าถูก

---

## Lab List

| Lab | ชื่อ | ชิป | ทดสอบอะไร |
|:---:|------|:---:|-----------|
| 01 | ไฟเลี้ยง + นาฬิกา | - | VCC/GND, กดปุ่ม clock |
| 02 | Ring Counter | U8, U24 | T0→T1→T2 วน |
| 03 | Program Counter | U1-U4 | นับ 0,1,2,3... |
| 04 | Address Mux | U15-U16 | เลือก PC หรือ IRL |
| 05 | ROM อ่านค่า | ROM, U7 | อ่าน byte จาก ROM |
| 06 | IR Latch | U5, U6 | จำคำสั่ง |
| 07 | ALU คำนวณ | U10-U13, U19-U20 | บวก ลบ XOR |
| 08 | AC เก็บผลลัพธ์ | U17-U18, U9 | ผลลัพธ์ → AC |
| 09 | Control Logic | U24-U28, U33 | สัญญาณควบคุม |
| 10 | ประกอบทั้งหมด | ทุกตัว | รันโปรแกรมจริง! |
