# Lab 02: Ring Counter — สร้างจังหวะ T0/T1/T2

**เป้าหมาย**: ต่อ U8 (74HC164) + U24 (74HC04) ให้สร้างจังหวะ 3 เฟส

---

## อุปกรณ์

| ชิ้น | ชื่อ | จำนวน |
|:----:|------|:-----:|
| 1 | 74HC164 (U8) | 1 |
| 2 | 74HC04 (U24) | 1 |
| 3 | LED สีเขียว 3mm | 3 |
| 4 | 330Ω resistor | 3 |
| 5 | 100nF capacitor | 2 |
| 6 | Push button (Reset) | 1 |
| 7 | 10kΩ resistor | 1 |

---

## วงจร

### Pinout: 74HC164 (U8)

```
        ┌───U───┐
   A  1 │       │ 14 VCC
   B  2 │       │ 13 Q7
  Q0  3 │ 164   │ 12 Q6
  Q1  4 │       │ 11 Q5
  Q2  5 │       │ 10 Q4
  Q3  6 │       │  9 /CLR
 GND  7 │       │  8 CLK
        └───────┘
```

### Pinout: 74HC04 (U24) — ใช้แค่ 2 gate

```
        ┌───U───┐
  1A  1 │       │ 14 VCC
  1Y  2 │       │ 13
  2A  3 │ 04    │ 12
  2Y  4 │       │ 11
     ...│       │...
 GND  7 │       │  8
        └───────┘
```

### การต่อสาย

```
1. ไฟเลี้ยง:
   U8 pin 14 (VCC) → 5V        U8 pin 7 (GND) → GND
   U24 pin 14 (VCC) → 5V       U24 pin 7 (GND) → GND
   100nF คร่อม VCC-GND ทั้ง 2 ตัว

2. Ring counter feedback:
   U8 pin 3 (Q0) → U24 pin 1 (1A)     [Q0 เข้า inverter 1]
   U24 pin 2 (1Y) → U8 pin 1 (A)      [NOT Q0 → A]
   
   U8 pin 4 (Q1) → U24 pin 3 (2A)     [Q1 เข้า inverter 2]
   U24 pin 4 (2Y) → U8 pin 2 (B)      [NOT Q1 → B]

3. Clock:
   U8 pin 8 (CLK) → สาย CLK จาก Lab 01 (ปุ่มกด)

4. Reset:
   U8 pin 9 (/CLR) → ปุ่ม Reset (ปกติ HIGH, กด = LOW)
   [ต่อ: 5V → 10kΩ → pin 9 ← ปุ่ม → GND]

5. LED ดู output:
   U8 pin 3 (Q0=T0) → 330Ω → LED → GND
   U8 pin 4 (Q1=T1) → 330Ω → LED → GND
   U8 pin 5 (Q2=T2) → 330Ω → LED → GND
```

---

## ทดสอบ ✅

### ขั้นตอน

1. เสียบไฟ → LED ทั้ง 3 อาจติดหรือดับ (ยังไม่ reset)
2. **กด Reset** → LED ทั้ง 3 ดับ (Q0=Q1=Q2=0)
3. **กด Clock ครั้งที่ 1** → T0 ติด, T1 ดับ, T2 ดับ
4. **กด Clock ครั้งที่ 2** → T0 ดับ, T1 ติด, T2 ดับ
5. **กด Clock ครั้งที่ 3** → T0 ดับ, T1 ดับ, T2 ติด
6. **กด Clock ครั้งที่ 4** → T0 ติด! (วนกลับมา!)

### ตาราง LED

| กด Clock | T0 (Q0) | T1 (Q1) | T2 (Q2) | ถูก? |
|:--------:|:-------:|:-------:|:-------:|:----:|
| Reset | ○ | ○ | ○ | ☐ |
| 1 | ● | ○ | ○ | ☐ |
| 2 | ○ | ● | ○ | ☐ |
| 3 | ○ | ○ | ● | ☐ |
| 4 | ● | ○ | ○ | ☐ |
| 5 | ○ | ● | ○ | ☐ |
| 6 | ○ | ○ | ● | ☐ |

● = LED ติด, ○ = LED ดับ

---

## ถ้าไม่ถูก?

| อาการ | สาเหตุ | แก้ |
|-------|--------|-----|
| LED ไม่ติดเลย | ไฟไม่ถึง IC | เช็ค VCC/GND |
| T0 ค้างติดตลอด | inverter ต่อผิด | เช็ค U24 pin 1→2, pin 3→4 |
| วนไม่ครบ 3 | A,B ต่อสลับ | สลับ U8 pin 1 กับ pin 2 |
| กด Reset แล้วไม่ clear | /CLR ไม่ถึง LOW | เช็คปุ่ม Reset |
