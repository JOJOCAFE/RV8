## ขั้นตอน

### 1. ต่อไฟเลี้ยง

```
USB 5V → ราง + (แดง) ด้านบน breadboard
USB GND → ราง - (ดำ) ด้านล่าง breadboard
```

**เช็ค**: วัดไฟระหว่างราง + กับ - ด้วยมัลติมิเตอร์ → ได้ ~5.0V

### 2. ต่อ LED ทดสอบไฟ (Power indicator)

```
ราง + (5V) → ตัวต้านทาน 330Ω → LED ขายาว(+) → LED ขาสั้น(-) → ราง - (GND)
```

**เช็ค**: LED ติดสว่าง → ✅ ไฟเข้าถูกต้อง

### 3. ต่อปุ่มกดเป็น Clock

```
ราง + (5V) → 10kΩ → จุด A ← ปุ่มกด → ราง - (GND)
                     │
                     └─→ 100nF → ราง - (GND)  [debounce]
                     │
                     └─→ สาย CLK (เอาไว้ใช้ Lab ถัดไป)
```

**วิธีทำงาน**:
- ไม่กด: จุด A = HIGH (5V ผ่าน 10kΩ)
- กดปุ่ม: จุด A = LOW (GND ผ่านปุ่ม)
- ปล่อย: กลับเป็น HIGH → **rising edge เกิดตรงนี้!**
- C 100nF ช่วยกรอง bounce

> ⚠️ **หมายเหตุสำคัญ: Clock Edge**
>
> ชิปส่วนใหญ่ใน RV8-GR (U1-U4, U5, U6, U8, U9) ทำงานตอน **rising edge**
> ดังนั้น CPU จะทำงาน **ตอนปล่อยปุ่ม** (LOW→HIGH) ไม่ใช่ตอนกดปุ่ม
>
> สรุป: **กดปุ่ม = เตรียม, ปล่อยปุ่ม = clock edge → CPU ทำงาน 1 ขั้น**
>
> (Lab ถัดไป ถ้าเพิ่ม 74HC14 Schmitt Trigger จะ invert สัญญาณให้
> กดปุ่ม = rising edge ได้ แต่สำหรับ lab นี้ใช้แบบนี้ก่อนก็ได้)

> 💡 **เรื่อง Debounce**
>
> RC (10kΩ + 100nF) ให้ τ ≈ 1ms ซึ่งพอสำหรับ lab นี้
> แต่ mechanical switch อาจ bounce ได้ 1-20ms
> ถ้าพบปัญหา "กด 1 ครั้ง นับ 2 ครั้ง" ใน Lab ถัดไป
> ให้เพิ่ม 74HC14 (Schmitt Trigger) เป็น debounce ที่ดีกว่า

### 4. ต่อ LED ดู Clock

```
จุด A (CLK) → ตัวต้านทาน 330Ω → LED สีเขียว → GND
```

| # | ทดสอบ | วิธีเช็ค | ผ่าน? |
|:-:|--------|----------|:-----:|
| 1 | ไฟ 5V ถึง breadboard | วัด multimeter = ~5V | ☐ |
| 2 | รางไฟต่อถึงกันตลอด | วัด continuity ปลายซ้าย↔ขวา | ☐ |
| 3 | LED ไฟแดง ติด | ดูด้วยตา | ☐ |
| 4 | กดปุ่ม → LED เขียวดับ | กดปุ่ม LED ดับ | ☐ |
| 5 | ปล่อยปุ่ม → LED เขียวติด | ปล่อย LED ติด | ☐ |
| 6 | กดปุ่มค้าง → LED ดับค้าง (ไม่กระพริบ) | debounce ดี | ☐ |
| 7 | กดแล้วปล่อยเร็ว → นับ 1 pulse เท่านั้น | ไม่ double-trigger | ☐ |

> ⚠️ **Breadboard หลายรุ่น: รางไฟขาดตรงกลาง!**
> ```
> + + + + +  [ขาด]  + + + + +
> ```
> วัดด้วย multimeter continuity ว่าปลายซ้าย↔ขวา ต่อถึงกัน
> ถ้าไม่ → ใช้สาย jumper bridge ข้ามตรงกลาง

> ⚠️ **Debounce ของ Lab นี้เป็นแบบเบื้องต้น**
> RC (10kΩ + 100nF) ช่วยลด bounce ได้ ~80%
> ถ้ายังเจอปัญหา "กด 1 ครั้ง นับ 2 ครั้ง" ใน Lab 02
> → เพิ่ม 74HC14 Schmitt Trigger จะแก้ได้ 100%

> ⚠️ **สำคัญ: Bypass Capacitor**
> ทุก IC ที่จะต่อใน Lab ถัดไป ต้องมี 100nF ceramic คร่อม VCC-GND ใกล้ขาชิป
> ถ้าไม่ใส่: 1 MHz อาจผ่าน แต่ 2 MHz จะ glitch สุ่ม

### Rising Edge — CPU ทำงานตอนไหน?

```
กดปุ่ม                     ปล่อยปุ่ม
   │                          │
   ▼                          ▼
   ┌──────────────────────────┐
───┘                          └───── CLK

LOW ◄── กดค้าง ──► HIGH

                              ↑
                         CPU ทำงาน!
                      (rising edge)
```

> 📌 CPU ทำงาน **ตอนปล่อยปุ่ม** (LOW→HIGH) ไม่ใช่ตอนกด!

## Troubleshooting

| อาการ | สาเหตุที่พบบ่อย |
|-------|----------------|
| LED แดงไม่ติด | ต่อ LED กลับขั้ว หรือ USB ไม่จ่ายไฟ |
| วัดได้ 0V ที่ราง | สาย USB เสีย หรือยังไม่เสียบ |
| วัดได้ 5V ด้านซ้าย แต่ 0V ด้านขวา | รางไฟขาดตรงกลาง → bridge |
| กดแล้ว LED เขียวไม่เปลี่ยน | ปุ่มเสีย หรือต่อผิดแถว |
| LED เขียวกระพริบมั่วตอนกด | Switch bounce → เพิ่ม 74HC14 ใน Lab 02 |
| Breadboard ร้อน | **ลัดวงจร! ถอดไฟทันที** ตรวจสาย VCC-GND |

> 💡 **เตรียมไว้สำหรับ Lab 02: Reset**
>
> Lab 02 (Ring Counter) ต้องมีปุ่ม Reset เพื่อเริ่มสถานะที่ถูกต้อง
> ถ้าอยากเตรียมล่วงหน้า สามารถต่อ Reset circuit ไว้ตอนนี้:
> ```
> ราง + (5V) → 10kΩ → จุด RST ← ปุ่ม Reset → ราง - (GND)
> จุด RST → 100nF → ราง - (GND)  [debounce]
> ```
> จุด RST = สาย /RST (ปกติ HIGH, กดปุ่ม = LOW = reset)
